<h2>◈ &nbsp;Featured Projects&nbsp; ◈</h2>
</div>

### 🔬 &nbsp;Student Face Biometric Attendance System

> *Computer Vision · Django · OpenCV · Deep Learning · Mobile Integration*

A production-grade **smart campus biometric engine** built from scratch during a hackathon. Detects, recognizes, and verifies student faces in real time via webcam or mobile camera — automatically marking attendance with login/logout timestamps.

- **Real-time face detection** using deep learning models with live video stream processing
- **Multi-device support** — works with mobile cameras and webcams out of the box
- **Admin dashboard** with session management, class-wise reports, and CSV export
- **Zero manual intervention** — fully automated attendance lifecycle
- Built in a hackathon environment, now extended into a scalable campus product

`Django` `OpenCV` `Deep Learning` `SQLite` `REST API` `Bootstrap`

---

### 🧠 &nbsp;DeepFake Detection Engine

> *Media Forensics · CNN · Transfer Learning · Binary Classification*

An AI-powered **media authenticity system** designed to detect AI-generated or manipulated video/image content with high accuracy. Trained on benchmark deepfake datasets using transfer learning architectures.

- **CNN-based classifier** fine-tuned on deepfake benchmark datasets
- **High-confidence binary classification** — Real vs. Fake with probability scores
- Designed with a modular pipeline for plug-in model upgrades
- Explores **Transformers and frequency-domain analysis** for next-gen detection

`Python` `PyTorch` `Transfer Learning` `OpenCV` `CNN` `Flask`

---

### 🛒 &nbsp;Django E-Commerce Platform

> *Full-Stack Engineering · Django · React · Payment Integration · REST API*

A full-featured, scalable **e-commerce application** with a Django backend powering a dynamic React frontend — built to production-standard architecture with authentication, product management, cart flow, and order tracking.

- **RESTful API architecture** with Django REST Framework
- Secure **user authentication**, role-based access, and session management
- Complete cart, checkout, and order lifecycle with payment gateway hooks
- Mobile-responsive UI with optimized load performance

`Django` `React` `DRF` `MySQL` `JWT Auth` `Bootstrap`
